### Обязательная часть (Easy)

**ДЗ-1: Поворот массива**

Поверни массив вправо на k позиций.

```
Input: arr = [1, 2, 3, 4, 5, 6, 7], k = 3
Output: [5, 6, 7, 1, 2, 3, 4]
```

Подсказка: Можно использовать три разворота!

---

**ДЗ-2: Слияние отсортированных массивов**

Даны два отсортированных массива. Объедини их в один отсортированный.

```
Input: arr1 = [1, 3, 5], arr2 = [2, 4, 6]
Output: [1, 2, 3, 4, 5, 6]
```

---

**ДЗ-3: Поиск дубликата**

Массив из n+1 чисел содержит числа от 1 до n. Найди дубликат.

```
Input: [1, 3, 4, 2, 2]
Output: 2
```

---

### Продвинутая часть (Medium)

**ДЗ-4: Минимальный размер подмассива**

Найди минимальную длину подмассива с суммой ≥ target.

```
Input: target = 7, arr = [2, 3, 1, 2, 4, 3]
Output: 2 ([4, 3])
```

---

**ДЗ-5: Подмассивы с суммой K**

Найди количество подмассивов с суммой равной K.

```
Input: arr = [1, 1, 1], k = 2
Output: 2
```

---

**ДЗ-6: Сортировка цветов (Dutch National Flag)**

Массив содержит только 0, 1 и 2. Отсортируй его in-place за один проход.

```
Input: [2, 0, 2, 1, 1, 0]
Output: [0, 0, 1, 1, 2, 2]
```

### Дополнительные задачи (для практики)

1. **Квадраты отсортированного массива** — вернуть массив квадратов в отсортированном порядке (массив может содержать отрицательные числа)

2. **Следующая перестановка** — найти следующую лексикографически большую перестановку

3. **Точка разворота** — найти индекс, где сумма слева равна сумме справа

4. **Максимальное расстояние до ближайшего нуля** — для каждой позиции найти расстояние до ближайшего нуля
